# 16 — Glossary

- **InboundFile**: raw upload από χρήστη.
- **ParsedDocument**: αποτέλεσμα parsing ενός inbound file.
- **Line Item**: μία γραμμή παραστατικού.
- **Normalization**: μετατροπή raw σε canonical.
- **Matching**: αντιστοίχιση line item σε προϊόν.
- **Provisional Product**: προσωρινό προϊόν που χρειάζεται enrichment.
- **Enrichment**: συμπλήρωση specs/κατηγοριών/attachments.
- **Provenance**: καταγραφή πηγής/χρόνου/χρήστη για κάθε τιμή.
- **Review Queue**: ουρά ανθρώπινου ελέγχου.
- **Hermes**: routing/orchestration layer για AI calls.
- **RAG**: retrieval augmented generation (semantic search πάνω σε docs).
